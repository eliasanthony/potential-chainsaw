Apple Cancellation

How to cancel a subscription on your iPhone or iPad

  1. Open the Settings app.

  2. Tap your name.

  3. Tap Subscriptions.

  4. Tap the subscription.

  5. Tap Cancel Subscription. You might need to scroll down to find the Cancel Subscription button. If there is no Cancel button or you see an expiration message in red text, the subscription is already canceled.

Understanding JIRA – JIRA Explained

How to build a JIRA? Explained.

(bolded items are template items/highlighted are to help you locate the info for the bolder items) JIRA templates can be found here: https://docs.google.com/document/d/1w_AzqYDlQKewb9nH5fvu4bWypNBBV7o2Xu0jZPKpDOk/edit

Description of the issue: In your notes/given to you by the customer

Email Address: In your notes/or from API

User state (Limited Commercial, Essential, Premium, Ex Subscriber): In API under account info at the top of the page or if a subscriber with a user package available you will enter the plan type (essential vs limited commercials vs premium)

Billing Platform: In API under user packages and source

Device ID: Comes from the customer directly from device (have customer open P+ app on device, go to settings, then support within the app) 

IP Address (https://whatismyipaddress.com/): Comes from the customer directly from device (have customer open P+ app on device, go to settings, then support within the app) or listed in user viewing history (Conviva) in API

Zip Code: API

Show/Season/Episode # (Please provide specific Show and Episodes): From Customer

Device Model & OS Version: From Customer (this is usually listed in the device settings about section)

App Version: Comes from the customer directly from device (have customer open P+ app on device, go to settings, then support within the app) or API user viewing history (Conviva)

Connection (Wi-Fi/Ethernet/Hotspot/Cellular): From Customer

Are you able to reproduce: This is a Yes if you can cause your system to have the same issue and No if you cannot (also please ensure to notate what device you used to replicate and if unable to replicate since you don’t have the same device, note this)

API URL: From Address Bar in browser while account is open in API (example below)

https://customer.paramountplus.com/customerProfile/USERID

Conviva Link: https://pulse.conviva.com/subscribers/?days=30&filter=4106&viewerId=USER ID Replace User ID in Coviva link with the number at end of the API link you just copy pasted above (above highlighted orange)

Troubleshooting Steps Taken: Add all steps performed with customer over the phone 

Date this started occurring: Customer will tell you when the issues started 

Speed Test results (for buffering/freezing issues; https://www.speedtest.net/): Always collect speed test results from website listed//Ensure you are collecting upload, download, and ping rates as these are required to help narrow down the issue
